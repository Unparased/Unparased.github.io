Chapter 1.2 Measures

Definition 1

μ : 𝒜 → [0, +∞] is said to be countably additive if and to be finitely additive if .

Then, a (countably additive) measure (or finitely additive measure) on 𝒜 is a function μ : 𝒜 → [0, +∞] that satisfies μ(⌀) = 0 and is countably (or finitely) additive.

【NOTE: In the following, the term measure is used to denote countably additive measure while the finitely additive measure will be called in full.】

Properties

Proposition 1.2.2 - Let (X, 𝒜, μ) be a measure space, and A, B are subsets of X that belong to 𝒜 and satisfy A ⊆ B. Then, μ(A) ≤ μ(B). If μ(A) < ∞, then μ(B − A) = μ(B) − μ(A).

Definition 2

μ is a finite measure if μ(X) < +∞ and σ-finite measure if X is the union of a sequence A1, A2, ... of sets belonging to 𝒜 and satisfy μ(Ai) < +∞ for each i.

  • Generally- A set in 𝒜 is σ-finite under μ if it is the union of a sequence of sets belonging to 𝒜 with finite measure.
  • The measure space (X, 𝒜, μ) is called finite or σ-finite if μ is finite or σ-finite.

Properties 2

Proposition 1.2.4 - Let (X, 𝒜, μ) be a measure space. If {Ak} is an arbitrary sequence of sets that belong to 𝒜, then Proposition 1.2.5 - Let (X, 𝒜, μ) be a measure space.

    1. If {Ak} is an increasing sequence of sets that belong to 𝒜, then μ(∪kAk) = limkμ(Ak).
    1. If {Ak} is a decreasing sequence of sets that belong to 𝒜, and if μ(An) < +∞ for some n, then μ(∩kAk) = limkμ(Ak).

Proposition 1.2.6 - Let (X, 𝒜) be a measurable space, and let μ be a finitely additive measure on (X, 𝒜). Then μ is a measure if either

    1. limkμ(Ak) = μ(∪kAk) holds for each increasing sequence {Ak} of sets that belong to 𝒜, or
    1. limkμ(Ak) = 0 holds for each decreasing sequence {Ak} of sets that belong to 𝒜 and satisfy ∩kAk = ⌀.

Terminology

  • A Borel measure on ℝd : A measure on (ℝd, ℬ(ℝd)).
  • A finite or σ-finite measure μ on (X, 𝒜) is continuous if μ({x}) = 0 holds for each x in X.
  • A finite or σ-finite measure μ on (X, 𝒜) is discrete if there is a countable subset D of X such that μ(Dc) = 0.
