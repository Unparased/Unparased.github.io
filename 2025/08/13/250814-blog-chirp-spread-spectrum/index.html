Paper 34 - An index-modulated FSCM systems with low complexity algorithm and enhanced data rate

FSCSS_IM

This waveform utilized the superimposed form by concurrently transmitting the symbols {f1, f2} with f1 < f2, implying ⌊log CM2⌋ bits.

ICSS_IM

This waveform is the enhanced version of FSCSS_IM, with f1 ≤ f2.

Detection

The detection relies on the DFT output of the received signal in the descending order, in which the difference between Y(1) − Y(2) and Y(2) − Y(3) is compared using a threshold T.

Channel Model

This channel model employed the doubly-selective channel, in which the delay shifts are assumed to be integers combined with the Doppler shifts to be fractionals. So the input-output relation can be directly obtained by using the conventional expression without the need for deriving a new equation.

Lemmas

This paper included some lemmas for deriving some relationship to simplify the expressions.

Message passing

This paper considered the message passing algorithm to enhance the performance combined with the truncated channel matrix. The truncation was induced by the weak contribution of some eigenvectors, further of the one of observations.
