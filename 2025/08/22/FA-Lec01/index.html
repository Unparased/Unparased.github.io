Normed Space

Definition 1 - A vector space V over a field 𝕂 (especially, ℝ or ℂ) is a set of vectors which comes with an addition + : V × V → V and scalar multiplication ⋅ : 𝕂 × V → V, along with some axioms: Commutativity, Associativity, Identity, and inverse of addition, identity of multiplication, and distributivity.

Axioms

  • Associativity of vector addition: u + (v + w) = u + v + w.
  • Commutativity of vector addition: u + v = v + u.
  • Identity Element of vector addition: ∃0 ∈ V s.t. 0 + v = v, ∀v ∈ V.
  • Inverse Element of vector addition: ∀v ∈ V, there exists −v ∈ V, s.t. v + (−v) = 0.
  • Compatibility of scalar multiplication with field multiplication: ∃0 ∈ V s.t. a(bv) = (ab)v.
  • Identity Element of scalar multiplication: ∃1 ∈ 𝕂 s.t. 1v = v, ∀v ∈ V.
  • Distributivity of scalar multiplication with respect to the vector addition: a(u + v) = au + av.
  • Distributivity of vector addition with respect to the scalar multiplication: (a + b)v = av + bv.

Examples

  • ℝn, ℂn, C([0, 1]).

Definition 2 - A vector space V is finite-dimensional if every linearly independent set is a finite set. In other words, for all sets E ⊆ V such that E has a finite cardinality. V is infinite-dimensional if it is not finite-dimensional.

Definition 3 - A norm on the vector space V is a function ‖⋅‖ : V → [0, ∞) satisfying the following three properties:

  • (Definiteness) ‖v‖ = 0 if and only if v = 0.
  • (Homogeneity) ‖λv‖ = |λ|‖v‖ for ∀v ∈ V and λ ∈ 𝕂.
  • (Triangle Inequality) ‖v1 + v2‖ ≤ ‖v1‖+‖v2‖ for ∀v1, v2 ∈ V.

A seminorm is a function ‖⋅‖ : V → [0, ∞) which satisfies (Homogeneity) and (Triangle Inequality) but not necessarily (Definiteness). A vector space equipped with a norm is called a normed space.

Definition 4 - A metric d : X × X → [0, ∞) is defined to follow the three conditions:

  • (Identification) d(u, v) = 0 if and only if u = v.
  • (Symmetry) d(u, v) = d(v, u).
  • (Triangle Inequality) d(u, v) + d(v, w) ≥ d(u, w) for all u, v, w ∈ V.
