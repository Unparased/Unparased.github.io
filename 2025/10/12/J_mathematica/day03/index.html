Defining variables

Expression Meaning
x = value assign a value to
x = y = value assign a value to and
x = . or Clear[x] remove any value assigned to
  • Remove values you assign to variables as soon as you finish using them.

Hint : Mathematica uses both upper- and lower-case letters.

Possible mistakes :

  • x y means x times y .
  • xy with no space is the variable with name xy .
  • 5x means 5 times x .
  • x^2y means (x^2) y , not x^(2y) .

Lists

Expression Meaning
{3, 5, 1}{3, 5, 1} a list of three numbers
{3, 5, 1} ^ 2 + 1{10, 26, 2} Operation on a list [entrywise]

Manipulating elements :

Expression Meaning
{a, b, c} a list
Part[list, i] or list[[i]] -th element of list [First- list[[1]]]
Part[list, {i, j, ...}] or list[[{i, j, ...}]] a list of -th, -th, …, elements of list
Part[list, i] = value or list[[i]] = value reset the -th element of list

Brackets

Brackets Meaning
(term) parenthesis for grouping
f [x] square brackets for functions
{a, b, c} curly braces for lists
v[[i]] double brackets for indexing

Sequence of operations

Expression Meaning
Expr1; Expr2; Expr3 give the result of the last expression
Expr1; Expr2; do expressions, but print no output [Inhibiting output]
