Hahn-Banach Theorem

DEF [Linear functional] - A functional 𝓁 : 𝒳 → ℝ is linear if ∀x, y ∈ 𝒳, α ∈ ℝ, then 𝓁(x + y) = 𝓁(x) + 𝓁(y) and 𝓁(αx) = α𝓁(x).

Theorem [HB] - Given 𝒴 ⊆ 𝒳 is a linear subspace over ℝ, define a linear functional 𝓁 : 𝒴 → ℝ. If 𝓅 : 𝒳 → ℝ satisfies the conditions:

  • [positive homogeneous] 𝓅(αx) = α𝓅(x) for ∀x ∈ 𝒳, α > 0,
  • [sub-additive] 𝓅(x + y) ≤ 𝓅(x) + 𝓅(y) for ∀x, y ∈ 𝒳,

and 𝓁(x) ≤ 𝓅(x) for ∀x ∈ 𝒴, then ∃ a functional ℒ : 𝒳 → ℝ such that (1) ℒ is linear, (2) ℒ(y) = 𝓁(y) for ∀y ∈ 𝒴, and (3) ℒ(x) ≤ 𝓅(x) for ∀x ∈ 𝒳.

  • proof : [Stage 1] Assume 𝒴 ≠ 𝒳, and ∃z ∉ 𝒴, 𝒴z = {y + αz : y ∈ 𝒴, α ∈ ℝ}.

    For y1 + α1z ∈ 𝒴z, y2 + α2z ∈ 𝒴z, y1 + α1z = y2 + α2z implies (α2 − α1)z = y1 − y2 ∈ 𝒴 such that α1 = α2 and y1 = y2.

    Define on 𝒴z. Then, ℒ has to satisfy (3), yielding that The first branch holds naturally. For the second branch, we have Therefore, a ≤ inf {𝓅(z + y) − 𝓁(y) : y ∈ 𝒴}. For the third branch, we have Therefore, a ≥ sup {𝓁(y) − 𝓅(y − z) : y ∈ 𝒴}. Then, we should prove sup {𝓁(y) − 𝓅(y − z) : y ∈ 𝒴} ≤ inf {𝓅(z + y) − 𝓁(y) : y ∈ 𝒴} ⇔ ∀y1, y2 ∈ 𝒴, 𝓁(y1) − 𝓅(y1 − z) ≤ 𝓅(z + y2) − 𝓁(y2).

    • proof : 𝓁(y1 + y2) = 𝓁(y1) + 𝓁(y2) ≤ 𝓅(y1 − z + (z + y2)) ≤ 𝓅(y1 − z) + 𝓅(z + y2). Then, 𝓁(y1) − 𝓅(y1 − z) ≤ 𝓅(z + y2) − 𝓁(y2), completing the proof.

    [Stage 2]Give Ω = {(𝒵, ℒ𝒵) : 𝒵 linear subspace of 𝒳, ℒ𝒵 satisfies three conditions}. Define < as (𝒵1, ℒ𝒵1) < (𝒵2, ℒ𝒵2) if 𝒵1 ⊆ 𝒵2 and ℒ𝒵2(z) = ℒ𝒵1(z) for z ∈ 𝒵1. Then, < is a partial order. By the hypothesis of Zorn’s lemma, ∃Ω′ ⊆ Ω is totally ordered. Ω′ = {(𝒵α, ℒ𝒵α) : α ∈ ℐ}. Then, Ω′ has an upper bound (𝒵, ℒ𝒵) with 𝒵 = ⋃α𝒵α and ℒ𝒵(z) = ℒ𝒵α(z) [well defined] for certain α such that z ∈ 𝒵α. Then, (𝒵, ℒ𝒵) is a maximal point of Ω. Then, 𝒵 = 𝒳[can be proved by contradiction].
