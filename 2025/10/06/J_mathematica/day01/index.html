Numerical Calculations

  • Basic expressions
Expression Meaning
x^y power
-x minus
x/y divide
x y z or x*y*z multiply
x+y+z add
  • Exact and approximate results

By default, Mathematica outputs the exact result as 2^100 = 1267650600228229401496703205376 rather than 1.26765*10^30. To output the approximate results, the input should become 2^100//N (N – numerical).

When the input has an explicit decimal point, the corresponding output becomes an approximate result.

  • Basic mathematical functions
Function Meaning
Sqrt[x] square root
Exp[x] exponential
Log[x] natural logarithm
Log[b,x] logarithm to base b
Sin[x], Cos[x], Tan[x] trigonometric functions
ArcSin[x], ArcCos[x], ArcTan[x] inverse trigonometric functions
n! factorial
Abs[x] absolute value
Round[x] closest integer
Mod[n,m] n modulo m
Random[] pseudorandom number [0, 1]
Max[x,y,...], Min[x,y,...] maximum and minimum
FactorInteger[n] prime factors
