Chapter 1.4 Lebesgue Measure

Proposition 1.4.1 [Regularity] - Let A be a Lebesgue measurable subset of ℝd. Then,

  • λ(A) = inf {λ(U) : U is open and A ⊆ U}, and
  • λ(A) = sup {λ(K) : K is compact and K ⊆ A}.

Lemma 1.4.2 [Property of half open cubes] - Each open subset of ℝd is the union of a countable disjoint collection of half-open cubes, each of which is of the form given by {(x1, …, xd) : ji2−k ≤ xi < (ji + 1)2−k for i = 1, …, d}. for some integers j1, …, jd and some positive integer k.

Proposition 1.4.3 - Lebesgue measure is the only measure on (ℝd, ℬ(ℝd)) that assigns to each d-dimensional interval, or even to each half-open cube its volume.

Definition 1 [Translate] - The translate of A ⊆ ℝd by x ∈ ℝd is A + x defined by A + x = {y ∈ ℝd : y = a + x for some a ∈ A}. Proposition 1.4.4 [Translation invariance] - Lebesgue outer measure on ℝd is translation invariant, in the sense that if x ∈ ℝd and A ⊆ ℝd, then λ*(A) = λ*(A + x). Furthermore, a subset B of ℝd is Lebesgue measurable iff B + x is Lebesgue measurable.

Proposition 1.4.5 - Let μ be a nonzero measure on (ℝd, ℬ(ℝd)) that is finite on the bounded Borel subsets of ℝd and is translation invariant, in the sense that μ(A) = μ(A + x) holds for each A in ℬ(ℝd) and each x in ℝd. Then there is a positive number c such that μ(A) = cλ(A) holds for each A in ℬ(ℝd).

Definition 2 [The Cantor Set] - Let K0 be the interval [0, 1]. Form K1 by removing from K0 the interval (1/3, 2/3). Continue this procedure, forming Kn by removing from Kn − 1 the open middle third of each of the intervals making up Kn − 1. The Cantor set is the set of points that remain; thus K = ∩nKn.

  • K is closed and bounded and has no interior points and Lebesgue measure 0.

  • The cardinality of K is that of the continuum.

    • sketch of proof : a sequence {zn} of 0's and 1's is assigned the number , which is a bijection of the set of all such sequences onto K.

      Employ the ternary representation: [Example] 1 = 1(0.22…); 0 = 0.00…; 1/3 = 0.1(0.022…); 2/3 = 0.2(0.122…).

      The initial interval K0 = [0, 1] followed by K1 = [0, 0.1] ∪ [0.2, 1]. Thus, the removal can be viewed to eliminate all numbers with the first digit after the radix point which is 1.[*]

To analyze the cardinality of all {zn}'s, we can construct the sequence as follows and assume it is countable: we can select a sequence {an} such that an ≠ znn. Proposition 1.4.7 - The Cantor set is a compact set that has the cardinality of the continuum but has Lebesgue measure of zero.

Theorem 1.4.9 - There is a subset of ℝ, and in fact of the interval (0, 1), that is not Lebesgue measurable.

  • proof : Define an equivalence relation ∼ satisfying x ∼ y ⇔ x − y ∈ ℚ. Based on the axiom of choice, we can form a subset E of (0, 1) whose elements are the representatives of all equivalent classes. Now we need to prove E is not Lebesgue measurable.

    Given a sequence {rn}, an enumeration of the rational numbers in (−1, 1), define En = E + rn. Hence,

    • En are disjoint,
    • ∪nEn is included in (−1, 2), and
    • (0, 1) is included in ∪nEn.

    Suppose E is Lebesgue measurable. λ(∪nEn) = ∑nλ(En). If λ(E) = 0, λ(∪nEn) = 0 contradicts λ(E) ≥ 1. On the other hand, if λ(E) > 0, λ(E) → ∞. Therefore, E is not Lebesgue measurable.

Definition 3 [Diff] - Diff(A) is the subset of ℝ defined by Diff(A) = {x − y : x ∈ A and y ∈ A}. Proposition 1.4.10 - Let A be a Lebesgue measurable subset of ℝ such that λ(A) > 0. Then Diff(A) includes an open interval that contains 0.

  • proof : There exists a open set U and a compact set K such that K ⊆ A ⊆ U and λ(U) < 2λ(K). Then, we can assert that the distance between points in K and the ones in Uc is positive and lower bounded. Therefore, ∃ε > 0, ∀|x| < ε, x + K ⊆ U. If x + K were disjoint from K, then 2λ(K) = λ(K) + λ(x + K) = λ(K ∪ (x + K)) ≤ λ(U), contradicting λ(U) < 2λ(K). Therefore, x ∈ Diff(K) ⇒ (−ε, ε) ⊆ Diff(K) ⊆ Diff(A).

Proposition 1.4.11 - There is a subset A of ℝ such that each Lebesgue measurable set that is included in A or in Ac has Lebesgue measure zero.

  • proof : Define subsets G, G0, and G1 of ℝ by Define a relation ∼ as x ∼ y ⇔ x − y ∈ G. Based on the axiom of choice, form a subset E of ℝ that contains exactly one representative of each equivalent class. Let A = E + G0.

    Assume there ∃ a subset B of A such that λ(B) > 0. Then, ∃ε > 0 (−ε, ε) ⊆ Diff(B) ⊆ Diff(A). G1 is dense in and intersects (−ε, ε), so there ∃x ∈ G1 and |x| < ε. Besides, x = e1 − e2 + g0 ∈ A where e1, e2 ∈ E ⇒ e1 − e2 ∈ E and g0 ∈ G0. Therefore, ∃g1 ∈ G1 such that g1 = e1 − e2 + g0 ⇒ e1 = e2, g1 = g0, leading to the contradiction. Therefore, A must have Lebesgue measurable set whose measure is zero.
