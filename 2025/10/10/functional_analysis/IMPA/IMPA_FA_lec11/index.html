Geometric Hahn-Banach Theorem

DEF [Hyperplane] - 𝓁 : 𝒳 → ℝ is linear and 𝓁 ≠ 0. Then, {x ∈ 𝒳 : 𝓁(x) = c} is a hyperplane with (closed) half-spaces {x ∈ 𝒳 : 𝓁(x) < ( ≤ )c} and {x ∈ 𝒳 : 𝓁(x) > ( ≥ )c}.

Theorem - 𝒦 ⊆ 𝒳 is a convex subset. All points x ∈ 𝒦 are interior, 𝒦 ≠ ⌀ and ∃y ∉ 𝒦, then ∃ℒ : 𝒳 → ℝ is linear and c ∈ ℝ such that 𝒦 ⊆ {x ∈ 𝒳 : ℒ(x) < c} and ℒ(y) = c.

  • proof : 0 ∈ 𝒦. Define 𝓅 = 𝓅𝒦, 0 where . x ∈ 𝒦 is interior so 𝓅(x) < 1. .

    Define 𝒴 = {αy : α ∈ ℝ} and 𝓁 : 𝒴 → ℝ is linear such that 𝓁(y) = 𝓅(y) ≥ 1 and 𝓁(αy) = α𝓁(y). Besides, 𝓁(αy) ≤ 𝓅(αy) on 𝒴.

    • proof : α = 0: 𝓁(αy) = 0 ≤ 𝓅(αy); α > 0: 𝓁(αy) = α𝓁(y) = α𝓅(y) = 𝓅(αy); 𝓁(αy) < 0 ≤ 𝓅(αy).

    Based on the Hahn-Banach Theorem, we can extend 𝓁 : 𝒴 → ℝ to ℒ : 𝒳 → ℝ such that ℒ(y) = 𝓁(y) = 𝓅(y) and ℒ(x) < ℒ(y) = c (ℒ(x) < 𝓅(x) ≤ 1 ≤ ℒ(y).). The proof is complete.

Corollary - 𝒦 is a convex set, ∃x ∈ 𝒦 is an interior point, ∃y ∉ 𝒦. Then, ∃ℒ : 𝒳 → ℝ, c ∈ ℝ and ℒ(y) = c, 𝒦 ⊆ {z ∈ 𝒳 : ℒ(z) ≤ c}.

Theorem - 𝒦1, 𝒦2 are convex sets, 𝒦1 ≠ ⌀, 𝒦2 ≠ ⌀. ∃x ∈ 𝒦1 is an interior point and 𝒦1⋂𝒦2 = ⌀. Then, ∃ℒ : 𝒳 → ℝ, c ∈ ℝ, ∀x ∈ 𝒦1, y ∈ 𝒦2, ℒ(x) ≤ c ≤ ℒ(y).

  • proof : Define 𝒦 = 𝒦1 − 𝒦2 = {x − y : x ∈ 𝒦1, y ∈ 𝒦2} which is a convex set. For some y ∈ 𝒦2, x − y is interior in 𝒦. 𝒦1⋂𝒦2 = ⌀ ⇒ 0 ∉ 𝒦.

    By the corollary, ∃ℒ : 𝒳 → ℝ such that ℒ(x1 − y1) ≤ 0 ⇒ ℒ(x1) ≤ ℒ(y1) for ∀x1 ∈ 𝒦1, y1 ∈ 𝒦2 where ℒ(0) = 0. Then, select c0 = sup {ℒ(x1) : x1 ∈ 𝒦1} such that ℒ(x1) ≤ c0 ≤ ℒ(y1). The proof is complete.

Applications

Case 1 - A normed linear space 𝒳 has a linear subspace 𝒴 ⊆ 𝒳. 𝓁 : 𝒴 → ℝ. ∃c0, |𝓁(y)| ≤ c0‖y‖, ∀y ∈ 𝒴. Then, ∃ℒ : 𝒳 → ℝ, ℒ(y) = 𝓁(y), ∀y ∈ 𝒴, |ℒ(x)| ≤ c0‖x‖, ∀x ∈ 𝒳. [norm is positive homogeneous and subadditive]
