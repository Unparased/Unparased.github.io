Normed Linear Spaces

DEF : A norm 𝒩 defined on a linear space 𝒳 is satisfying the following properties:

  • 𝒩(x) ≥ 0 for ∀x ∈ 𝒳, 𝒩(x) = 0 ⇔ x = 0.
  • α ∈ 𝕂, x ∈ 𝒳, 𝒩(αx) = |α|𝒩(x).
  • x, y ∈ 𝒳, 𝒩(x + y) ≤ 𝒩(x) + 𝒩(y).

OBS : |‖x‖−‖y‖| ≤ ‖x − y‖, ‖ − x‖ = ‖x‖.

DEF : A metric d defined as d : 𝒳 × 𝒳 → ℝ+ by d(x, y) = ‖x − y‖ with

  • d(x, y) ≥ 0, d(x, y) = 0 ⇔ x = y.
  • d(x, y) = d(y, x).
  • d(x, z) ≤ d(x, y) + d(y, z).
