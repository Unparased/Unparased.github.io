Linear Spaces

DEF : 𝒳 is a linear space over a field 𝕂 (ℂ or ℝ) if 𝒳 is a set with 2 operations + and ⋅ (scalar multiplication).

  • +: x, y ∈ 𝒳 ⇒ x + y ∈ 𝒳
    • Commutative: x + y = y + x
    • Associative: x + (y + z) = x + y + z
    • Group: ∃0: 0 + x = x; ∃−x: −x + x = 0
  • ⋅ : x ∈ 𝒳, α, β ∈ 𝕂 ⇒ αx ∈ 𝒳
    • Commutative: (αβ)x = α(βx)
    • Distributive: (α + β)x = αx + βx, α(x + y) = αx + αy
    • 1: 1 ⋅ x = x

DEF : 𝒴 ⊆ 𝒳 is a linear subspace if x, y ∈ 𝒴, α ∈ 𝕂 such that αx + y ∈ 𝒴.

  • {𝒴θ : θ ∈ ℐ} is a set including linear subspace of 𝒳, then ∩θ ∈ ℐ𝒴θ is a linear subspace of 𝒳.

DEF : {𝒴θ, θ ∈ ℐ} is totally ordered if θ1, θ2 ∈ ℐ 𝒴θ1 ⊆ 𝒴θ2 or 𝒴θ1 ⊇ 𝒴θ2.

  • {𝒴θ : θ ∈ ℐ} is a totally ordered set including linear subspace of 𝒳, ∪θ ∈ ℐ𝒴θ is a linear space.

DEF : 𝒮 ⊆ 𝒳, linear span of 𝒮 is LS(𝒮) = ∩θ𝒴θ for 𝒴θ ⊇ 𝒮. [Smallest linear space containing S]
