Quotient Spaces

DEF : For a linear subspace 𝒴 ∈ 𝒳, x ∼ y if x − y ∈ 𝒴. [Equivalence]

  • x ∼ x
  • x ∼ y ⇒ y ∼ x
  • x ∼ y, y ∼ z ⇒ x ∼ z

DEF : (𝒳 modulo 𝒴) 𝒳|𝒴 = {[x] = {y ∈ 𝒴 : y ∼ x} : x ∈ 𝒳}.

  • [x] + [y] = [x + y]

DEF : (Linear maps) 𝒯 : 𝒳 → 𝒳* is a linear map if 1) 𝒯(x + y) = 𝒯(x) + 𝒯(y) and 2) 𝒯(αx) = α𝒯(x).

  • 𝒴 ⊆ 𝒳 is linear subspace, and then 𝒯(𝒴) = {𝒯(x) : x ∈ 𝒴} is a linear subspace.
  • 𝒴* ⊆ 𝒳* is a linear subspace, and then 𝒯−1(𝒴*) = {x ∈ 𝒳 : 𝒯(x) ∈ 𝒴*} is a linear space.

DEF : (Isomorph) 𝒳 and 𝒳* are isomorphic if ∃𝒯 : 𝒳 → 𝒳* bijection.

DEF : (Convex) 𝒦 ∈ 𝒳 is convex if x, y ∈ 𝒦, α ∈ [0, 1], αx + (1 − α)y ∈ 𝒦.

  • 𝒴1, 𝒴2 ∈ 𝒳 are convex subsets, and then 𝒴1 + 𝒴2 is convex.
  • 𝒦θ, θ ∈ ℐ are convex, and then ∩θ𝒦θ is convex.
  • 𝒦θ, θ ∈ ℐ are convex and totally ordered, and then ∪θ𝒦θ is convex.
  • 𝒦 is convex $$ 𝒯(𝒦) is convex.
  • 𝒦* is convex $$ 𝒯−1(𝒦*) is convex.

DEF : (Convex hull) 𝒮 ⊆ 𝒳, CU(𝒮) = ∩θ𝒦θ for all convex sets 𝒦θ ⊇ 𝒮. [Smallest convex set containing S]
