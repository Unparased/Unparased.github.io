Hahn-Banach Theorem

DEF [Linear functional] - A functional 𝓁 : 𝒳 → ℝ is linear if ∀x, y ∈ 𝒳, α ∈ ℝ, then 𝓁(x + y) = 𝓁(x) + 𝓁(y) and 𝓁(αx) = α𝓁(x).

Theorem [HB] - Given 𝒴 ⊆ 𝒳 is a linear subspace over ℝ, define a linear functional 𝓁 : 𝒴 → ℝ. If 𝓅 : 𝒳 → ℝ satisfies the conditions:

  • [positive homogeneous] 𝓅(αx) = α𝓅(x) for ∀x ∈ 𝒳, α > 0,
  • [sub-additive] 𝓅(x + y) ≤ 𝓅(x) + 𝓅(y) for ∀x, y ∈ 𝒳,

and 𝓁(x) ≤ 𝓅(x) for ∀x ∈ 𝒴, then ∃ a functional ℒ : 𝒳 → ℝ such that (1) ℒ is linear, (2) ℒ(y) = 𝓁(y) for ∀y ∈ 𝒴, and (3) ℒ(x) ≤ 𝓅(x) for ∀x ∈ 𝒳.

  • proof : [Stage 1] Assume 𝒴 ≠ 𝒳, and ∃z ∉ 𝒴, 𝒴z = {y + αz : y ∈ 𝒴, α ∈ ℝ}.

Zorn’s lemma

DEF [Partial order] - For a linear space 𝒳, define a (total) partial order ℛ following the conditions:

  • Reflexive : ∀x ∈ 𝒳, xℛx.
  • Anti-symmetric : ∀x, y ∈ 𝒳, xℛy, yℛx ⇒ x = y.
  • Transitive : ∀x, y, z ∈ 𝒳, xℛy, yℛz ⇒ xℛz.
  • [TOTAL] : ∀x, y ∈ 𝒳, either xℛy or yℛx.

Exm - (x1, x2), (y1, y2) ∈ ℝ2, then define an order < as (x1, x2) < (y1, y2) if x1 ≤ y1 and x2 ≤ y2. [Partial order NOT TOTAL ORDER]

DEF [Upper bound] - A partial order (𝒳,  < ). Given A ⊆ 𝒳, 𝒵 is an upper bound for A if x ∈ 𝒵 for ∀x ∈ A.

DEF [Maximal element] - A ⊆ 𝒳, z ∈ A is a maximal element if y ∈ A, z < y ⇒ z = y.

Lemma [Zorn’s] - Given (𝒳,  < ) and 𝒳 ≠ ⌀, assume any totally ordered subset 𝒴 of 𝒳 admits upper bound. Then 𝒳 has a maximal element.

Exm - ℝ2 cannot hold.

DEF [Base] - {Xθ : θ ∈ ℐ} is a base of 𝒳 if

  • ∀N, ∀θ1, …, θN ∈ ℐ ⇒ {Xθ1, …, XθN} is linearly independent.
  • 𝒳 = span{Xθ : θ ∈ ℐ}.

Theorem - For a linear space 𝒳 ≠ ⌀, ∃{Xθ : θ ∈ ℐ} a base of 𝒳.

  • proof : Define (Ω,  < ). for ∀N, ∀θ1, …, θN ∈ ℐ where Xθ1, …, XθN are linearly independent. {Xθ : θ ∈ ℐ} < {Yθ′ : θ′ ∈ ℐ′} if ∀θ ∈ ℐ, ∃θ′ ∈ ℐ′ such that Xθ = Yθ′.

Proposition 1.4.3 - Lebesgue measure is the only measure on (ℝd, ℬ(ℝd)) that assigns to each d-dimensional interval, or even to each half-open cube its volume.

Definition 1 [Translate] - The translate of A ⊆ ℝd by x ∈ ℝd is A + x defined by A + x = {y ∈ ℝd : y = a + x for some a ∈ A}. Proposition 1.4.4 [Translation invariance] - Lebesgue outer measure on ℝd is translation invariant, in the sense that if x ∈ ℝd and A ⊆ ℝd, then λ*(A) = λ*(A + x). Definition 2 [The Cantor Set] - Let K0 be the interval [0, 1]. Form K1 by removing from K0 the interval (1/3, 2/3). Continue this procedure, forming Kn by removing from Kn − 1 the open middle third of each of the intervals making up Kn − 1. The Cantor set is the set of points that remain; thus K = ∩nKn.
